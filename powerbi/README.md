# Internet Sales Dashboard (Power BI)

## Overview
An interactive **Power BI dashboard** built on the **Microsoft AdventureWorks dataset (2024–2026)** to analyse internet sales performance across customers, products, and time.

The dashboard focuses on **user-driven exploration**, allowing sales teams to quickly drill into high-value customers, best-selling products, and specific periods.
